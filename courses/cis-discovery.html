5.Which ServiceNow release has introduced the possibility of migrating probes to patterns?
  • ANew York
  • BOrlando
  • CParis
  • DQuebec
Source: DocumentationThe correct answer is: New YorkExplanation:The release of New York in ServiceNow introduced the capability to migrate probes to patterns. Prior to the New York release, ServiceNow used probes for discovery, which were scripts specifically written for each discovery target. However, with the New York release, ServiceNow introduced patterns as a more flexible and scalable approach to discovery.Patterns allow for a more modular and reusable way to define discovery behavior. They are collections of probe modules, sensors, and identification rules that can be applied to multiple target devices or systems.

9.Where is the Shazzam probe placed at the beginning of horizontal discovery?

  • BIn the Pattern Log.
  • CIn a request in the ECC queue.
  • DIn the Discovery Log.
Source: DocumentationThe correct answer is "In a request in the ECC queue".Explanation:In ServiceNow, the ECC (Enterprise Control Center) Queue is a table that stores messages and data transferred between a MID Server and the ServiceNow instance. When a discovery is initiated, the Shazzam probe (a powerful and flexible probe that can run any command on a target device) is placed in a request in the ECC Queue to be processed by a MID Server.You can think of the ECC Queue like a post office and the Shazzam probe like a letter. When you want to send a letter (probe), you drop it off at the post office (ECC Queue).
