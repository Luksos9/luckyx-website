1.What is the primary purpose of the Hardware Asset Management (HAM) application?
  • ATo manage software licenses
  • BTo track the lifecycle of physical hardware assets
  • CTo configure network devices
  • DTo manage employee onboarding
HAM tracks hardware assets through their complete lifecycle — from procurement through deployment, maintenance, and retirement/disposal.
2.Which table stores hardware asset records in ServiceNow?
  • Acmdb_ci_hardware
  • Balm_hardware
  • Cast_hardware
  • Dcmdb_hardware
The alm_hardware table is the primary table that stores hardware asset records in the ServiceNow Asset Management application.
3.What is a Stockroom in HAM?
  • AA virtual container for software licenses
  • BA physical or logical location where assets are stored
  • CA type of asset report
  • DA configuration item class
A Stockroom represents a physical or logical location where hardware assets are stored, such as a warehouse, IT closet, or shipping dock.
4.What is the purpose of Model Categories in HAM?
  • ATo classify assets by manufacturer
  • BTo group models into categories for lifecycle management and reporting
  • CTo define warranty terms
  • DTo set depreciation rates
Model Categories group product models into logical categories (e.g., Computer, Printer, Server) to standardize lifecycle processes and enable category-specific reporting.
5.Which lifecycle stage indicates an asset is actively being used?
  • AOn Order
  • BIn Stock
  • CIn Use
  • DRetired
The 'In Use' lifecycle stage indicates that a hardware asset has been deployed and is actively being used by an end user or for a business function.
