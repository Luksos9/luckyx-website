5.What happens if the skills configuration option is set to &quot;all&quot; and no agents possess all the required skills for a task?
  • ANone are auto-assigned
  • BThe task is assigned to the first available agent
  • CAuto-assignment occurs without considering skills match
  • DThe agent with the closest skills match is auto-assigned
0) Source ServiceNow Product Documentation: Work order task assignment skills Note: The "All" setting acts as a strict "AND" operator.1) Correct Answer None are auto-assigned2) Detailed Expert Explanation This is a strict logic gate in the assignment engine. If a Task requires Skills A, B, and C, and the configuration is set to "All" (Mandatory), the query effectively says: SELECT Agent WHERE Skills CONTAINS (A AND B AND C)If Agent 1 has A and B (but not C), they fail the check. If no agent has all three, the result set is Empty. Consequently, None are auto-assigned.
